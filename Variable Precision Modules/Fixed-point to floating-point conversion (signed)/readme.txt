This module is the signed fix point to floating point conversion module.
To use this module, please find the general modules folder and add the following modules to the project:
parameterized_absolute_value, parameterized_priority_encoder, parameterized_subtractor, parameterized_variable_shifter, parameterized_adder, parameterized_mux, parameterized_or_gate.
